This is a Community Contribution game.

For this challenge, two leagues are available. There are no changes between the leagues, except that your opponents have already proven a higher level of skills.

  The Goal

Battleship is a turn-based game, where the objective is to find and sink the opponents ships before the opponent sink yours.

  Rules

The game is played is played on a grid field 10x10. The top left corner is A1.
The bottom right corner is J10.

Initialisation

The first move is an initialization of the board in placing the ships.
The ships will be placed in this exact order: Carrier(size:5) Battleship(size:4) Destroyer (size:3) Submarine (size:3) Patrol Boat (size:2).
The initialization command contains the coordinate of the ship (Top Left) and the orientation (H)orizontal (V)ertical, separated by ";".
The ships must be placed inside the grid and are not allowed to intersect each other.

Example: A1V;B1V;C1V;D1V;E1H

Seeking

Each turn you get the result of the previous guess and your opponents guess. If there had not been a previous guess, you get "-" as result.

The result could be (M)iss (H)it or (S)unk in case the ship has no hit points anymore, separated by ";" and the guess of your opponent.

Example: H;A1

In your turn, you guess one field to bomb like A1 (The result will be provided by the referee, with the opponents next move).

The player who sinks the opponents ships first, or has made most points win.

  Game Input

Initialization Input
Line 1: String value can be ignored

Initialization Output
Line 1: ships A1V;B1;C1V;D1V;E1H five ships in order of size (5,4,3,3,2) and their orientation separated with ";"
Input for one game turn
Line 1: responseM;A3 One single string containing the result of your previous guess and your opponents guess, separated by ";"

Output for one game turn
Line 1: guessA1 One single field from the grid, like A1, B3, C7...
Victory Conditions
  • You sink all the opponents ships.
  • At the end of all turns have done more points than your opponent.
Loss Conditions
  • You initialisation is wrong by:
    - Invalid command (Coordinate (Must be UPPERCASE), Orientation (H,V only valid), Separator (;only valid))
    - Invalid number of ships
    - ships placed outside the grid (Coordinate)
    - ships intersect on placement
  • Your guess is wrong by:
    - Invalid command
    - guess is outside the grid (Coordinate)
